# Atlas – Reference Knowledge Index

The atlas contains stable, reusable reference knowledge that accumulates across all projects.
Unlike project notes, atlas content is not project-specific – it documents methods, datasets,
infrastructure patterns, and architectural decisions that apply broadly.

## Sections

| Directory | What lives here |
|-----------|----------------|
| methods/ | Computational methods, algorithms, ML techniques, tools |
| datasets/ | Dataset documentation, characteristics, preprocessing notes |
| infrastructure/ | Pod setup, HPC cluster, Kubernetes, CERIT infrastructure |
| decisions/ | Cross-project architectural decision records |

## Write rules
- Atlas content is **stable** – only update when your understanding changes
- Add new notes sparingly; prefer updating existing notes over creating new ones
- Always cross-link: if a method is used by a project, the project notes should reference the atlas entry
- The librarian agent manages atlas writes – do not write here without explicit instruction
